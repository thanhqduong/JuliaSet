Julia Set Fractal

$$\text{Consider the dynamical system } f: \mathbb{C} \rightarrow \mathbb{C} \text{ given by} $$ $$f(z) = z^2 + c \text{ for } c \in \mathbb{C} \text{ with } |c| < 2.$$ $$\text{The set of initial conditions } z_0 \text{ for which the solution } (x_n) \text{ is bounded is call the Filled Julia Set for }f.$$

The Julia Set is the boundary of the Filled Julia Set.

Change the c value by moving the cursor inside the box below to see the fractals!
